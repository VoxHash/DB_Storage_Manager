# Support

## Getting Help

- **Documentation**: Check [README.md](README.md) and [docs/](docs/) for guides
- **Issues**: Open an issue using the appropriate template in [GitHub Issues](https://github.com/voxhash/db-storage-manager/issues)
- **Discussions**: Join [GitHub Discussions](https://github.com/voxhash/db-storage-manager/discussions) for questions and ideas
- **Contact**: Email contact@voxhash.dev

## Before Reporting

1. Check existing issues and discussions
2. Review documentation and FAQ
3. Ensure you're using the latest version
4. Provide detailed information about your issue

## Issue Templates

- **Bug Report**: Use the bug report template for issues
- **Feature Request**: Use the feature request template for new ideas
- **Documentation**: Use the docs improvement template for documentation issues
